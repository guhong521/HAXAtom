## 资源池 vs 预设方案关系图（Lego Model）

```
                ┌─────────────────────────────────────┐
                │      资源池（Resource Pools）        │
                │  （原子化能力仓库 / Lego 零件库）     │
                └─────────────────────────────────────┘
        ┌───────────┬───────────┬───────────┬───────────┬───────────┐
        │   模型池   │  提示词池  │   插件池   │  知识库池  │   记忆池   │
        │Model Pool │Prompt Pool│Plugin Pool│  KB Pool  │Memory Pool│
        └───────────┴───────────┴───────────┴───────────┴───────────┘
              │           │           │           │           │
              │           │           │           │           │
              ▼           ▼           ▼           ▼           ▼

┌─────────────────────────────────────────────────────────────────────┐
│                         预设方案（Preset）                           │
│        （从资源池挑选零件 → 组合成一个完整的 AI 智能体实例）         │
│                                                                     │
│   selected_model            → 引用模型池的某个模型                   │
│   selected_prompt           → 引用提示词池的人设/系统提示词          │
│   selected_plugins          → 引用插件池的工具列表                   │
│   selected_knowledge_bases  → 引用知识库池                           │
│   selected_memory           → 引用记忆池                             │
│                                                                     │
│   overrides                 → 当前方案的专属覆盖参数（不修改资源池）  │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              运行时智能体实例（Runtime Agent Instance）              │
│           （由预设方案 + 资源池动态组装而成，完全无硬编码）          │
└─────────────────────────────────────────────────────────────────────┘
```

## 预设方案加载流程图（Preset Engine Flow）

```
                      用户选择预设方案
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│  1. 读取预设方案（Preset）                                           │
│     - 校验 preset_id 存在且已激活                                    │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│  2. 校验所有引用的资源ID                                             │
│     - 模型ID      → 模型池                                           │
│     - 提示词ID    → 提示词池                                         │
│     - 插件ID列表  → 插件池                                           │
│     - 知识库ID    → 知识库池                                         │
│     - 记忆ID      → 记忆池                                           │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│  3. 从资源池中拉取原子化配置                                         │
│     - 模型配置（ModelConfig）                                        │
│     - 提示词配置（PromptConfig）                                     │
│     - 插件配置（PluginConfig）                                       │
│     - 知识库配置（KnowledgeBase）                                    │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│  4. 应用 overrides 覆盖参数                                          │
│     - temperature / top_p / max_tokens 等模型参数                    │
│     - 仅对当前对话生效，不修改资源池原始配置                         │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│  5. 组装 LangGraph StateGraph 工作流                                 │
│                                                                     │
│     ┌─────────────┐    ┌─────────────┐    ┌─────────────┐          │
│     │  retrieval  │───→│    agent    │───→│    tools    │          │
│     │  (RAG节点)   │    │ (Agent节点)  │←───│ (ToolNode)  │          │
│     └─────────────┘    └──────┬──────┘    └─────────────┘          │
│                               │                                     │
│                               ▼                                     │
│                             ┌──────┐                                │
│                             │ END  │                                │
│                             └──────┘                                │
│                                                                     │
│     技术栈：LangGraph StateGraph + LCEL（LangChain Expression Language）│
│     - 支持循环调用工具                                               │
│     - 支持状态持久化（MemorySaver）                                  │
│     - 支持条件分支（Conditional Edges）                              │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│  6. 生成运行时智能体实例                                             │
│     - StateGraph.compile() → 可执行图                               │
│     - 绑定 checkpointer 支持对话记忆                                 │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
                      用户开始对话（流式输出）
```

## LangGraph 工作流详解

```
┌─────────────────────────────────────────────────────────────────────┐
│                    LangGraph StateGraph 结构                         │
└─────────────────────────────────────────────────────────────────────┘

  用户输入
      │
      ▼
┌─────────────────────────────────────────────────────────────────────┐
│  State: AgentState                                                   │
│  {                                                                    │
│    messages: Sequence[BaseMessage],    # 对话历史                    │
│    input: str,                         # 当前输入                    │
│    preset_id: str,                     # 预设ID                      │
│    session_id: str,                    # 会话ID                      │
│    retrieved_context: str,             # RAG检索结果                 │
│    needs_retrieval: bool,              # 是否需要检索                │
│    tool_results: List[Dict],           # 工具执行结果                │
│    needs_tool_call: bool,              # 是否需要调用工具            │
│    step_count: int,                    # 步骤计数                    │
│    max_steps: int,                     # 最大步骤限制                │
│    overrides: Dict,                    # 覆盖参数                    │
│    error: Optional[str]                # 错误信息                    │
│  }                                                                    │
└─────────────────────────────────────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────────────────────────────────────┐
│  Node: retrieval（可选）                                             │
│  - 条件：enable_rag=True 且存在知识库                                │
│  - 功能：执行向量检索，获取相关文档                                  │
│  - 输出：更新 state.retrieved_context                               │
└─────────────────────────────────────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────────────────────────────────────┐
│  Node: agent                                                         │
│  - LCEL链：ChatPromptTemplate | BaseChatModel                       │
│  - 输入：messages（包含system prompt + history + user input）        │
│  - 输出：AIMessage（可能包含 tool_calls）                            │
│  - 功能：模型推理，决定是否需要调用工具                              │
└─────────────────────────────────────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────────────────────────────────────┐
│  Conditional Edge: should_continue_to_tools                         │
│  - 检查 AIMessage 是否包含 tool_calls                               │
│  - 是 → 路由到 tools 节点                                            │
│  - 否 → 路由到 END                                                   │
└─────────────────────────────────────────────────────────────────────┘
      │                           │
      │ 有 tool_calls              │ 无 tool_calls
      ▼                           ▼
┌─────────────────────────────────┐  ┌─────────────────────────────────┐
│  Node: tools                     │  │  END                            │
│  - LangGraph ToolNode            │  │  - 返回最终 AI 回复              │
│  - 执行所有请求的工具            │  │  - 保存对话历史                  │
│  - 返回 ToolMessage 列表         │  │  - 流式输出给前端                │
└─────────────────────────────────┘  └─────────────────────────────────┘
      │
      ▼
  返回 agent 节点（循环）
```

## RAG 检索链路图（Retrieval Flow）

```
┌─────────────────────────────────────────────────────────────────────┐
│              知识库池（Knowledge Base Pool）                         │
│                                                                       │
│  kb_id        │ collection_name │ embedding_model │ retrieval_params │
│  ─────────────┼─────────────────┼─────────────────┼──────────────────│
│  docs_kb      │ kb_docs         │ bge-small-zh    │ top_k=5          │
│  code_kb      │ kb_code         │ text-embedding  │ top_k=10         │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              文档上传（PDF / TXT / MD / DOCX）                       │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              文档切片（Text Splitter）                               │
│                                                                       │
│  - chunk_size: 500~1000 字符                                         │
│  - chunk_overlap: 50~100 字符                                        │
│  - 使用 RecursiveCharacterTextSplitter                               │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              向量化（Embedding Model）                               │
│                                                                       │
│  - BGE-Small-ZH（中文，本地）                                        │
│  - Text-Embedding-3-Large（OpenAI）                                  │
│  - 其他 Embedding 模型                                               │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              ChromaDB（向量数据库）                                  │
│                                                                       │
│  - collection_name = kb_{kb_id}                                     │
│  - 持久化存储（data/chroma/）                                        │
│  - 支持元数据过滤                                                    │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              RAG 检索引擎（RAGEngine）                               │
│                                                                       │
│  1. 接收用户查询 query                                               │
│  2. 查询向量化（使用知识库配置的 embedding_model）                   │
│  3. 多知识库并行检索                                                 │
│  4. 相似度排序 + 阈值过滤                                            │
│  5. 格式化上下文（format_retrieved_context）                         │
└─────────────────────────────────────────────────────────────────────┘

                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              LCEL 检索链（可选组件）                                 │
│                                                                       │
│  - Query Rewriting（查询重写，多轮对话场景）                         │
│  - Context Compression（上下文压缩）                                 │
│  - Re-ranking（重排序，提升精度）                                    │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              注入系统提示词                                          │
│                                                                       │
│  "基于以下参考信息回答问题：\n\n{retrieved_context}\n\n用户问题：..." │
```

## 技术栈汇总

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **Web框架** | FastAPI + Uvicorn | 高性能异步Web框架 |
| **工作流引擎** | LangGraph StateGraph | 支持循环、分支、持久化的Agent编排 |
| **链式语法** | LCEL (LangChain Expression Language) | 声明式链定义，支持流式输出 |
| **LLM调用** | LangChain ChatModels | 统一接口支持多厂商 |
| **向量存储** | ChromaDB | 本地向量数据库，支持持久化 |
| **文本分割** | LangChain Text Splitters | 支持多种分割策略 |
| **工具调用** | LangGraph ToolNode | 原生支持工具调用循环 |
| **记忆系统** | LangGraph MemorySaver | 支持短期/长期记忆 |
| **数据库** | SQLAlchemy + aiosqlite | 异步ORM，支持SQLite |
| **配置管理** | Pydantic Settings | 类型安全的配置管理 |
| **流式输出** | SSE (Server-Sent Events) | 实时推送AI回复 |
